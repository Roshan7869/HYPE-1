# HYPE — We're Hiring

**Project:** HYPE — India's first live auction marketplace for hype culture
**Stack:** Next.js 14 + Tailwind + framer-motion + TypeScript
**Location:** Remote (India preferred, Mumbai-based a plus)
**Status:** Three open roles

---

## Role 1: Senior Frontend Engineer (Next.js)

**Focus:** Component architecture, API integration, performance optimization

### Responsibilities
- Build and maintain page-level components across 32 routes
- Implement SWR-based data fetching patterns (hooks under `src/api/`)
- Wire frontend to 40+ API endpoints
- Optimize LCP (< 1.5s), CLS (< 0.05), INP (< 200ms)
- Build the 3-click buyer flow (Home → Bid → Pay)

### Requirements
- 3+ years React, 1+ years Next.js App Router
- TypeScript, Tailwind, SWR/React Query
- Experience with framer-motion or similar animation libraries
- Understanding of Core Web Vitals and image optimization
- Nice to have: recharts, lenis, auction/live-bidding UIs

### What you'll work on first
1. Wire the "Place Bid" handler (currently broken)
2. Connect cart and checkout to real data (currently mock)
3. Build the API client layer (SWR hooks)
4. Optimize route-level code splitting
